# Real Ranker Project

## Project Setup

### Clone Repository:

```
git clone https://github.com/jvatencia/real-ranker
```

### Install dependencies

```
cd frontend && npm install
```

### Create .env file and add firebase credentials

```
REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_AUTH_DOMAIN=
```

### Run App

```
npm run start
```
